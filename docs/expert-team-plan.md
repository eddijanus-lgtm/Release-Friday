# Release Friday expert team hardening

This branch is executed as five coordinated roles:

1. Product lead — keeps the Thursday-to-Friday release workflow focused and removes misleading states.
2. Music data engineer — combines MusicBrainz, Apple/iTunes storefront search and curated confirmations.
3. Frontend engineer — connects real covers and streaming links and makes all states data-driven.
4. Product designer — preserves the approved Figma visual language while improving contrast and empty states.
5. QA engineer — verifies mobile and desktop click-throughs, filters, saving, search, external links and empty data.

The branch is complete only when build and automated browser QA pass.
